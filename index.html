🎭 임시 확인 모드

텀블벅 후원 목록을 붙여넣기만 하면 바로 확인할 수 있어요.
데이터는 브라우저 메모리에만 저장되고, 새로고침하면 사라져요.
나중에 시트에 연결하면 계속 관리할 수 있어요.

1구글 시트 만들기

  1. sheets.google.com 접속 → + 빈 스프레드시트 클릭
  2. 상단 메뉴에서 확장 프로그램 → Apps Script 클릭
  3. 새 탭에서 Apps Script 편집기가 열려요

2코드 붙여넣기

  1. 편집기에서 Code.gs 열어서 기존 코드 전부 지우기 (Ctrl+A → Del)
  2. 아래 Code.gs 복사 버튼 눌러서 붙여넣고 💾 저장
내용 펼쳐보기 

            

          

          
    
            
  1. 왼쪽 파일 목록에서 + → 스크립트 → 이름 SheetSetup 입력
    2. 
            
  2. 기본 코드 지우고 아래 SheetSetup.gs 복사 버튼 눌러서 붙여넣고 💾 저장
    3. 
          

          

            
            

              내용 펼쳐보기
              

            

          

        


        

          
3시트 초기화 실행

          
    
            
  1. 상단 함수 드롭다운 → setupSheet 선택 → ▶ 실행
    2. 
            
  2. 첫 실행 시 권한 검토 창이 뜸: 권한 검토 → 본인 계정 선택 → 고급 → 프로젝트로 이동 → 허용
    3. 
            
  3. ⚠️ 경고는 구글이 아직 검증 안 한 개인 스크립트라서 뜨는 것. 본인이 만든 거니 안전해요
    4. 
            
  4. 실행 완료 후 시트에 projects 탭과 헤더 16개가 생겼는지 확인
    5. 
          

        


        

          
4웹앱으로 배포

          
    
            
  1. Apps Script 편집기 오른쪽 위 배포 → 새 배포
    2. 
            
  2. ⚙ 톱니 아이콘 → 웹 앱 선택
    3. 
            
  3. 설정:
              
      
                
    • 다음 사용자로 실행: 나
      • 
                
    • 액세스 권한이 있는 사용자: 모든 사용자
      • 
              
    
            
    4. 
            
  4. 🔒 "모든 사용자"로 설정해야 브라우저 접근이 가능해요. URL을 아는 사람만 쓸 수 있으니 걱정 말아요
    5. 
            
  5. 배포 클릭 → 권한 허용 → 웹 앱 URL 나옴 → 복사
    6. 
          

        


        

          
5연결 완료

          
    
            
  1. 위의 빠른 연결 탭으로 돌아가서 복사한 URL 붙여넣기
    2. 
            
  2. 연결하기 클릭 → 끝! 🎉
    3. 
          

          

            
          

        


        

          
💡 팁

          
    
            
  • 문제가 생기면 SheetSetup.gs의 checkSheetStatus() 실행
    • 
            
  • 카테고리는 카드에서 태그 클릭으로 바로 추가/변경 가능
    • 
            
  • 키보드 단축키: N(추가), /(검색), S(선택 모드)
    • 
            
  • 텀블벅 "내 후원 현황" 페이지를 전체 복사해서 "붙여넣기로 추가"하면 자동으로 정리됨

          사용 방법
          
    
            
  1. 텀블벅 로그인 → 프로필 → 후원 현황(후원한 프로젝트) 열기
    2. 
            
  2. 후원 목록 영역을 드래그해서 복사
    3. 
            
  3. 아래 칸에 붙여넣기 → 분석하기
    4. 
          

          
💡 한 번에 안 되면 프로젝트별로 나눠 여러 번 붙여넣어도 돼요. 중복은 자동으로 걸러집니다.
