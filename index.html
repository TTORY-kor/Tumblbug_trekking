사용 방법
  1. 텀블벅 로그인 → 프로필 → 후원 현황(후원한 프로젝트) 열기
  2. 후원 목록 영역을 드래그해서 복사 (Ctrl+C / ⌘+C)
  3. 아래 칸에 붙여넣기 (Ctrl+V / ⌘+V) → 분석

💡 복사가 잘 안 되면 한 프로젝트씩 분리해서 여러 번 붙여넣어도 돼요. 중복은 자동으로 걸러집니다.
